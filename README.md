# 微信增强插件 WechatEnhancement
本项目需要Xposed，同时支持[VirtualXposed](https://github.com/android-hacker/VirtualXposed) 

目前支持抢红包，自动接收转账，消息防撤回，朋友圈防删除，朋友圈去广告

自适应方式采用微信巫师的自动搜索方式，理论支持微信全版本，微信升级后理论上可继续支持
